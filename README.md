# MageBot

![RaccoonLabs](https://img.shields.io/badge/Built%20by-RaccoonLabs-blueviolet)

A modular Discord bot with AI-powered commands and server management.

## What It Does
MageBot is a feature-rich Discord bot that combines utility commands, moderation tools, and AI-powered interactions. It supports custom plugins, slash commands, and integrates with various APIs for a seamless server experience.

## Tech Stack
- Python 3.10+
- discord.py, aiohttp

## Quick Start
```bash
git clone https://github.com/CyberRaccoonTeam/magebot.git
cd magebot
pip install -r requirements.txt
export DISCORD_TOKEN=your_token
python bot.py
```
